## API

除了以下参数，其余和 [`antd InputNumber`](https://ant.design/components/input-number-cn/) 组件一样。

参数 | 说明 | 类型 | 默认值 |
------------- | ------------- | ------------- | ------------- |
before  | 输入框前面元素 | `ReactNode` | - |
after  | 输入框后面元素 | `ReactNode` | - |
